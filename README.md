### 简介

提供你本人的任何信息，只要信息量充足、你使用大语言模型上下文足够长并且其模态支持这些信息类型，本SKILL就能为你生成一个专属于你的数字克隆体副本.SKILL，供你将该该副本分发给他人，或与自己灵魂共鸣，或作陪伴，或作社交拓展。

### 使用方法

#### 一、本SKILL使用方法

- 构建产物新产物：`/create [项目名称]`

### 提醒

- 本SKILL在构建产物SKILL时只认同信息数据流，对用户的任何描述和要求均不予参考和录入，以确保用户不会生成虚假的、不符合现实自己人格的产物SKILL。
- 产物SKILL本身是有状态的，每一次与产物SKILL的交互都会改变产物SKILL本身，请确保在构建完初始构建产物SKILL后保留其初始副本。
- 允许在构建产物SKILL是要求产物SKILL无状态，即每一次与产物SKILL的交互都不会改变产物SKILL本身。
- 如果需要构建产物SKILL中只包含部分现实人格，请确保录入的数据仅包含该部分现实人格的内容。
